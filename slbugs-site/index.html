Serverless Bugs
Welcome to Serverless Bugs! My name is Brian Bates and this web application is a serverless bug tracker built with React, Apollo GraphQL, utilizing Amazon Web Services, specifically AWS Lambda, DynamoDB, API Gateway, and Cognito.

To get started, click the button below to register an account. You will recieve an email with a confirmation code for verification.
NOTE: This website, and its accompanying site app.slbugs.com, are a part of a portfolio project of mine. You can view the code by clicking the github button below.
Please do not use this application for any serious efforts as I will be continuously altering the design and backend.
